_Изменения для v 7.9.2_:
- `FS`: реализована возможность выгрузки считанных статусов ФН в ту же директорию, в которую сохраняются выгрузки `.fnc` и `.fnm`;
- `FS`: в интерфейсе кэширования выгрузок добавлена подстановка текущей директории сохранения файлов `.fnc` в качестве предлагаемой для добавления новых ссылок в базу;
- `PR`: список полей, обрабатываемый ядром приложения, теперь полностью и корректно изолирован от интерфейса;
- `EC`: добавлена возможность временного отключения отслеживания без удаления записи о ККТ;
- `PR Android`: реализована совместимая версия приложения для ОС Android;
- `PR Windows`: скорректировано поведение некоторых элементов интерфейса;
- `PR`: исправлены неточности в форматах файлов заявлений;
- `PR`: в предлагаемое имя файла добавлено время формирования заявления (для исключения наложения файлов, формируемых группой для одного пользователя);
- `PR`: скорректировано отображение списка вариантов работы с ОФД для заявления о первичной регистрации;
- `PR`: отключено получение из ФН номера ФД отчёта о перерегистрации, если выбрана причина «замена ФН»;
- `PR`: интерфейсная часть теперь корректно отвязана от ядра приложения;
- `PR`: в заявлениях реализована кодировка `UTF8`;
- `EC`: модифицирован внешний вид интерфейса выбора ККТ из списка;
- `EC`: скорректировано поведение программы при пустом списке отслеживания
