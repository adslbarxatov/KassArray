_Изменения для v 7.6.8_:
- `Windows`: исправлен сброс выбранного протокола в разделе команд нижнего уровня при выходе из программы;
- `ФН`: отправка документов ОФД теперь корректно пропускается, если в последней регистрации в ФН есть признак автономного режима;
-` ФН`: автоматический запуск отправки документов ОФД после из записи в ФН теперь можно отключить (например, в случаях формирования чеков датой, более старой, чем текущая, чтобы избежать ошибки «Неверная разница между двумя датами»);
- `ФН`: улучшен алгоритм записи документов в ФН, убраны лишние диалоговые окна;
- Дополнен список команд нижнего уровня для протокола `Штрих`;
- `ФН`: реализована возможность подмены текущих даты и времени записи в ФН (по умолчанию берутся из ОС) более старыми значениями (например, для решения задач смены ОФД при открытой смене и истёкшем ресурсе хранения ФД);
- `ФН`: реализована возможность подмены даты, указанной в шаблоне (пере)регистрации, текущей датой (из ОС или настройки, описанной выше);
- Добавлена модель ККТ `Life Pulse S10F` из приказа от 11.12.25;
- Разделено описание моделей, исключённых из реестра, и моделей, подлежащих исключению при определённых условиях;
- Добавлено описание ошибки `0x3447 (241)` – неверная разница между датами;
- `ФН`: добавлена поддержка ставок НДС `22%` и `22/122` на чтение и запись
