- `ФН`: реализована подстановка текущей даты с шагом в одну минуту между чеками в режиме пакетного проведения: при включении подмены даты, указанные в шаблоне, будут замещены расчётными значениями на момент запуска записи в ФН;
- Уточнены сведения о ряде моделей ККТ, включая версии ПО для `РР-02` и `MSPOS-T-Ф` из приказа от 26.11.25;
- К описаниям моделей ККТ добавлены пометки о типах устройств и возможности автономной работы;
- `КЗ`: реализована подсистема агрегации реквизитов пользователей ККТ для быстрой подстановки при формировании последующих заявлений. Добавление выполняется при успешной проверке реквизитов перед отправкой задания на печать;
- Уточнена серия ЗН и другие параметры для ККТ модели `POSCENTER 02Ф`;
- Актуализирован перечень ТС ПИоТ;
- Исправлены некоторые проблемы в поведении поиска моделей ККТ;
- `ФН`: обновлён интерфейс записи документов в ФН;
- `ФН`: добавлена печатная форма для тега `1192` (дополнительный реквизит чека, рекомендованный к использованию при коррекции расчётов по предписаниям ФНС)
