_Изменения для v 7.12_:
- `FS`: проведена экспериментальная проверка корректности выгрузки УРМТ (5694 уведомления) с загрузкой в ЛК ЧЗ: файл обработан без ошибок, формат корректен;
- `FS`: расширение файла выгрузки УРМТ изменено на `.crpt2` для соответствия интерфейсу ЛК ЧЗ;
- `FS`: для выгрузок ФН, сохраняемых в формате CSV, добавлено экранирование формата в полях ФПД и ФПО, чтобы длинные значения никогда не представлялись в экспоненциальной форме;
- `HC`: добавлены новые модели ТС ПИоТ;
- `EC Android`: функционал продукта теперь полностью поддерживается Android-версией;
- `EC Windows`: реализовано корректное перемещение курсора в списке ККТ при изменении позиции ККТ, данные которой были обновлены, или при добавлении новой ККТ;
- Обновлены пакеты поддержки `.NET`;
- `EC`: для владельцев ККТ, у которых не задан ИНН, в главном окне добавлена соответствующая пометка;
- `EC`: скорректирован контроль наличия ИНН у всех владельцев в списке ККТ перед экспортом данных для обмена между приложениями;
- `PR`: приложение теперь запоминает изменения, внесённые при заполнении заявления в данные об организации, даже если она ранее уже была добавлена, и для неё уже работает быстрая подстановка;
- Восстановлены (в очередной раз) сведения о ТС ПИоТ для ККТ `Ритейл` и `POSCenter`;
- `EC`: унифицирована идентификация владельцев ККТ – ключом списка теперь является ИНН, указание которого теперь обязательно;
- `EC`: реализовано визуальное различие между случаями, когда состояние тарифа ОФД неизвестно и когда ККТ работает без ОФД;
- `EC`: реализовано полноценное редактирование реквизитов владельцев ККТ;
- `EC`: реализован импорт и экспорт данных ККТ между инстанциями приложения;
- `FS`: добавлено ручное кэширование файлов статусов ФН
